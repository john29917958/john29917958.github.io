Actually, this character was born because I want to make an assistant who can help me schedule my life, giving me reminders, so I never lost important life events. However, I found that [Google Assistant](https://assistant.google.com/) is yet powerful enough and... I really do not have much time for such side project (to beat Google !? :P). Maybe I should think of another goal for my Anassa project, so I can have the differency from the Google Assistant ;)

## Character Evolution
Yeah I admit... I take UI and art very seriously ^^". Since the origin customer of this project is me, so the Anassa App must includes a beautiful character ha ha. Be careful, here is the first version of Anassa...

[ArtStation link](https://tzuchaowang.artstation.com/projects/PA0b8)
{% asset_img 2018-07-29_first_anassa_design.jpg First Anassa design %}

Yuck! Right? The body shape is terrible and the armor is not beautiful. I think the main contribution of this version of Anassa is that, I set the main concept to technology look. The theme is decided to be cyan since I want to combine intelligence, reliability (blue) and vitality (green) together.

[ArtStation link](https://tzuchaowang.artstation.com/projects/aKqQ9)
{% asset_img 2018-09-23_second_anassa_design.jpg Second Anassa design %}

Here's the second version of Anassa. In this versin, I started to search for anime style body shapes and take them as reference. As you can see... It works XD. The proprtion of entire body is better now. In addition to body shapes, I collected some artworks with technical look as well, so the character now includes more elements with technology feel. Here is the sketch of the 2nd. Anassa design:
{% asset_img 2018-09-23_second_anassa_design_concept.jpg Second Anassa design %}

[ArtStation link](https://tzuchaowang.artstation.com/projects/xzDABO)
{% asset_img last_anassa_design.jpg Last Anassa design %}

This is the latest version of Anassa character. I made this version since I want Anassa to be more... well... like a normal girl @@ The previous version of Anassa is more like a robot and is somehow not that approachable and adorable. So I changed her useless body armor into a dress with some patterns (it took me some time to design the pattern = ="). The useless wings changed to translucent mini skirt. Hard and cold armors on the feet changed to black stockings. Face is now cuter, too! Also, the color set was reselected. I lowers the saturation of hair color so the entier look of character is more comfortable. To keep the technical look, I added additional wings and gems to feet and hands.
