感謝您參與「Ha」遊戲封閉測試，測試方式如下：
1. 到 App Store 下載並安裝 [TestFlight](https://apps.apple.com/tw/app/testflight/id899247664)
{% asset_img test-flight-icon.png %}
2. 打開 TestFlight，第一次使用畫面會呈現「沒有可測試 App」
{% asset_img test-flight-no-app.jpg %}
3. 到您的 e-mail 信箱打開 TestFlight 寄出的測試邀請信件
{% asset_img mail-title.jpg %}
4. 按下藍色的 TestFlight 按鈕
{% asset_img e-mail-app.jpg %}
5. Ha 遊戲應用程式會出現，按下「打開」按鈕
{% asset_img install.jpg %}
6. 大功告成，開始測試
{% asset_img icon.jpg %}
